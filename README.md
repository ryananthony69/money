# Infinite Money Glitch UGC Video

This repository contains a GitHub Actions workflow that turns the supplied Methods app screenshots and screen recording into a polished 9:16 UGC advertisement.

The workflow generates:

- neural narration with an offline fallback;
- animated Methods app screenshots and screen-recording cuts;
- dynamic burned-in captions;
- an original royalty-free music bed;
- a 1080×1920 H.264/AAC MP4, thumbnail, transcript, and render report;
- an on-screen AI-generated and illustrative-earnings disclosure.

Open a pull request against `master` or run **Generate Infinite Money Glitch UGC Video** manually. The result is uploaded as the `infinite-money-glitch-ugc` Actions artifact for 30 days.
